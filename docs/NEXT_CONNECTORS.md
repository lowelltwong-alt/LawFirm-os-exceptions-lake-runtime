# Next Connectors

These are future connector candidates only. They are not implemented in this MVP.

Potential future connectors:

- Litify

- client or matter portals
- AR systems

- Excel-based intake surfaces

Future connector work must remain outside this MVP unless it preserves the same core boundaries:

- no canon mutation
- no schema redefinition
- no lifecycle redefinition
- no promotion-authority redefinition
- no real firm operational data in synthetic test fixtures

This MVP intentionally ships with zero connector code.
